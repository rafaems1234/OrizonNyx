## Rota Brasil - Dashboard de Análise de Atrasos Logísticos 

# 📋 Sobre o Projeto

O Rota Brasil é um dashboard desenvolvido para analisar e monitorar atrasos em entregas logísticas, permitindo a visualização de indicadores de desempenho por transportadora e região.

O objetivo do projeto é fornecer uma visão clara dos gargalos operacionais, auxiliando gestores na tomada de decisão e na melhoria dos processos logísticos.


# 🚀 Funcionalidades

* Visualização de indicadores de desempenho.

* Análise de atrasos por transportadora.

* Análise de atrasos por região.

* Ranking de desempenho.

* Filtros para navegação dos dados.

* Dashboard responsivo e intuitivo.





# 🛠️ Tecnologias Utilizadas

* HTML5

* CSS3

* JavaScript

* GitHub Pages





# 📊 Metodologia

Os atrasos foram identificados por meio da comparação entre a data prevista e a data real de conclusão das entregas.

O cálculo foi realizado utilizando:

atraso = diasReais - prazo

Valores negativos foram ajustados para zero, garantindo que apenas atrasos efetivos fossem considerados nas análises.

Os dados foram agrupados por transportadora e região geográfica para geração de indicadores, rankings e gráficos.
